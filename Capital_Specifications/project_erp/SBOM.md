## 5. 组件详细信息

### 5.1 agentos_core

**描述**：AgentOS 核心内核模块，提供基础系统功能。

**主要功能**：
- 错误处理（error.c）
- 进程间通信（IPC）
- 内存管理
- 任务调度
- 时间管理

**文件结构**：
- `include/` - 公共头文件
- `src/` - 源代码

### 5.2 agentos_coreloopthree

**描述**：核心循环模块，集成认知、执行和记忆引擎。

**主要功能**：
- 认知引擎：处理用户输入，生成任务计划
- 执行引擎：执行任务，监控执行状态
- 记忆引擎：存储和检索记忆
- 核心循环：协调各引擎工作

**文件结构**：
- `include/` - 公共头文件
- `src/cognition/` - 认知引擎源代码
- `src/execution/` - 执行引擎源代码
- `src/memory/` - 记忆引擎源代码

### 5.3 agentos_syscall

**描述**：系统调用模块，提供统一的系统接口。

**主要功能**：
- 任务管理：提交、查询、等待和取消任务
- 记忆管理：写入、搜索、获取和删除记忆
- 会话管理：创建、获取、关闭和列出会话
- 可观测性：获取系统指标和追踪数据

**文件结构**：
- `include/` - 公共头文件
- `src/` - 源代码
- `tests/` - 测试代码

### 5.4 agentos_memoryrovol

**描述**：记忆卷模块，提供分层记忆存储和检索。

**主要功能**：
- 原始存储层：存储原始记忆数据
- 特征提取层：提取记忆特征
- 结构组织层：组织记忆结构
- 模式挖掘层：挖掘记忆模式

**文件结构**：
- `include/` - 公共头文件
- `src/` - 源代码
- `tests/` - 测试代码

### 5.5 agentos_cupolas

**描述**：安全域模块，提供安全隔离和权限管理。

**主要功能**：
- 虚拟工位：隔离执行环境
- 权限裁决：基于规则的权限检查
- 审计：记录系统操作
- 输入净化：净化用户输入

**文件结构**：
- `include/` - 公共头文件
- `src/` - 源代码
- `tests/` - 测试代码

### 5.6 agentos_dynamic

**描述**：动态模块，提供运行时管理和外部接口。

**主要功能**：
- HTTP 网关：处理 HTTP 请求
- WebSocket 网关：处理 WebSocket 连接
- stdio 网关：处理标准输入输出
- 健康检查：监控系统健康状态
- 可观测性：提供系统指标

**文件结构**：
- `include/` - 公共头文件
- `src/` - 源代码

## 8. 构建信息

### 8.1 构建系统

- **构建工具**：CMake 3.20+
- **构建类型**：Debug/Release
- **编译选项**：-Wall -Wextra -Werror -O2
- **链接选项**：根据依赖项自动设置

### 8.2 构建步骤

1. 配置构建目录：`cmake -B build`
2. 编译项目：`cmake --build build`
3. 运行测试：`cmake --build build --target test`
4. 安装项目：`cmake --build build --target install`
